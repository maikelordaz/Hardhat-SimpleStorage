# Hardhat - Storage

This is a basic project to demonstrate how to make deploys, tasks, and tests with hardhat

## :rocket: INSTALLATION

1. Clone this repo
2. Install the dependencies with 
    > npm install
    > yarn install
3. Run test with any of
    > npx hardhat test
    > yarn hardhat test
